## 🛠️ Stack Tecnológico

- **Frontend**: Angular 16+
- **UI**: Bootstrap 5 + ng-bootstrap
- **Testing**: Jasmine + Karma
- **Gestor de Paquetes**: npm

## 🚀 Primeros Pasos

```bash
git clone https://github.com/moonslayers/sys-2-credit-frontend
cd sys-2-credit-frontend
npm install
ng serve
```
